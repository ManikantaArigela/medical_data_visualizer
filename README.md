# medical_data_visualizer


This project is part of the freeCodeCamp Data Analysis with Python certification.

The project uses:
- Python
- Pandas
- Matplotlib
- Seaborn
- NumPy

The goal of this project is to analyze medical examination data and visualize relationships between:
- cardiovascular disease
- cholesterol
- glucose
- smoking
- alcohol intake
- physical activity
- overweight condition

---

# Project Features

## 1. Categorical Plot

The categorical plot shows:
- counts of good and bad outcomes
- grouped by cardio condition
- for:
  - cholesterol
  - gluc
  - smoke
  - alco
  - active
  - overweight
